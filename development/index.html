Ravencoin integration guide for developers

The blockchain is a list of transactions that outputs coins and or assets.

Overview

Building something for/on Ravencoin most likely mean that you want to read and write data

  • Start with 12 words (the wallet)
  • Based on the 12 words, derive/generate addresses
  • Check the balance of the addresses on the blockchain
  • To send/transfer
    • You first need to get all the Unspent transaction output (UTXOs) for you addresses
    • Filter out enough UTXOs for the transaction
    • Use rpc call createrawtransaction to create a transaction spending the given inputs and creating new outputs. Use your "enough" UTXOs as input
    • Sign the raw transaction
    • Broadcast the raw transaction
Raven Rebels have NPM packages for all of these steps
